# 📊 Customer Churn Analysis | Power BI Project

## 🚀 Project Overview

This project focuses on analyzing customer churn behavior and identifying high-risk customers using **Power BI**. The goal is to uncover actionable insights that help businesses improve customer retention and reduce churn.

---

## 🎯 Key Objectives

* Analyze customer churn patterns
* Identify high-risk customers likely to leave
* Build an interactive dashboard for decision-making
* Provide data-driven retention strategies

---

## 🛠️ Tools & Technologies Used

* **Power BI Desktop**
* **Power Query (Data Cleaning & Transformation)**
* **DAX (Data Analysis Expressions)**
* **Microsoft Excel**

---

## 📂 Dataset

* Publicly available **Customer Churn Dataset**
* Used for educational and analytical purposes

---

## 🔍 Project Workflow

### 1️⃣ Data Cleaning & Preparation

* Removed null and inconsistent values
* Transformed data using Power Query
* Standardized columns for analysis

### 2️⃣ Data Modeling & DAX

* Created calculated columns and measures
* Key metrics:

  * Churn Rate
  * Active Customers
  * Customer Segmentation

### 3️⃣ Dashboard Development

* Designed interactive Power BI dashboard
* KPI cards for quick insights
* Visualizations:

  * Customer churn distribution
  * Demographic insights
  * Subscription behavior

### 4️⃣ High-Risk Customer Identification

* Segmented customers based on churn probability
* Highlighted key risk indicators

---

## 📈 Key Insights

* Identified major factors contributing to customer churn
* Certain customer segments show higher churn probability
* Usage patterns and subscription types influence retention

---

## 💡 Business Recommendations

* Target high-risk customers with personalized offers
* Improve engagement strategies
* Optimize pricing and subscription plans
